# Neural Traffic Prediction System

This repository contains the project files for my dissertation, *Neural Traffic Prediction System*.

## Notebooks
- 01_data_inspection.ipynb – initial inspection of the dataset
- 02_data_cleaning.ipynb – cleaning and preprocessing
- 03_feature_engineering.ipynb – feature creation
- 04_baseline_model.ipynb – baseline model
- 05_random_forest_model.ipynb – random forest model
- 06_lstm_model.ipynb – LSTM model
- 07_hotspot_detection.ipynb – hotspot detection
- 09_routing_prototype.ipynb – routing prototype
